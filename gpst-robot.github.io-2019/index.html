成员介绍

2014~2018

Association of course | 协会小讲堂

在课余时间，协会为成员们提供专业化知识培训和有规划的课外活动，以此来提高协会成员的综合能力。协会尽最大的努力让成员了解更多，学到更多，充实自己的大学生活。

Interests and Hobbies | 兴趣和爱好

在协会里可以让对编程、电子和机械感兴趣的小伙伴，一起共同探究兴趣爱好。

C 语言

C语言是一门面向过程、抽象化的通用程序设计语言，广泛应用于底层开发。

Arduino

Arduino是一款便捷灵活、方便上手的开源电子原型平台，包含硬件和软件。

raspberry Pi

Raspberry Pi(中文名为“树莓派”,简写为RPi，只有信用卡大小的微型电脑，其系统基于Linux。

机械臂

机械臂是指高精度，高速点胶机器手，机械臂是一个多输入多输出、高度非线性、强耦合的复杂系统。

Equipment | 设备

协会内拥有各类型机器人：仿人机器人，自制轮式机器人，大疆无人机，Dobot魔术师，树莓派，Arduino等。

仿人机器人 Darwin op2

达尔文具有先进的计算能力，先进的传感器，高负载能力和动态运动能力，可实现许多令人兴奋的研究，教育和外展活动。

树莓派

别看其外表“娇小”，内“心”却很强大，视频、音频等功能通通皆有。

自制万向轮机器人

这是由协会成员自制的万向轮机器人，可实现水果采摘等多功能操作。

大疆无人机

在无人机系统,手持影像系统与机器人教育领域成为业内领先的品牌。

Dobot魔术师

DOBOT机械臂可以灵活翻转挪移，能进行画、写、移动、抓握东西。
